README
Проєкт: Модуль лояльності продуктового магазину
Застосунок: LoyaltyManager
Мова програмування: C#
Тип застосунку: Windows Forms
База даних: SQLite

1. Опис проєкту
Даний застосунок призначений для автоматизації обліку клієнтів
та нарахування бонусних балів у продуктовому магазині.
Система дозволяє авторизуватися користувачу, переглядати
список товарів, здійснювати покупки та накопичувати бонусні бали.

2. Запуск проєкту
Для запуску проєкту необхідно:
- операційна система Windows 10 або новіша;
- встановлене середовище Visual Studio 2022;
- встановлений .NET 6.0 або новіший.

Для запуску:
1) Відкрити файл LoyaltyManager.sln у Visual Studio.
2) Натиснути кнопку "Start" (F5).
3) Після запуску відкриється форма авторизації.

3. База даних
База даних SQLite створюється автоматично при першому запуску
застосунку. Файл бази даних:
- LoyaltyDB.db

Файл бази даних зберігається у каталозі запуску застосунку
(папка з виконуваним файлом).

Структура бази даних включає таблиці:
- users
- products
- purchases

4. Дані для входу (тестові облікові записи)
Для входу в систему можна використати наступні логіни:

Логін: admin
Пароль: 1234
Роль: адміністратор

Логін: user1
Пароль: 1234
Роль: користувач

Логін: user2
Пароль: 1234
Роль: користувач

5. Основний функціонал
- авторизація користувачів;
- перегляд списку товарів;
- здійснення покупки товару;
- автоматичне нарахування бонусних балів;
- перегляд історії покупок;
- відображення поточного балансу бонусів.

6. Особливості роботи
- бонусні бали нараховуються автоматично відповідно до кількості товару;
- усі операції зберігаються у базі даних;
- інтерфейс користувача реалізований українською мовою;
- при виході з облікового запису застосунок перезапускається.
